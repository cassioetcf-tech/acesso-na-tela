Declaração de Acessibilidade

O Acesso na Tela é um portal sobre cinema e streaming acessíveis. Nosso compromisso com a acessibilidade não se limita ao conteúdo que publicamos — ele se estende à forma como construímos o próprio site.

Nível de conformidade

Este site busca conformidade com as Diretrizes de Acessibilidade para Conteúdo Web (WCAG) 2.1, nível AA, publicadas pelo World Wide Web Consortium (W3C).

Estamos em processo contínuo de melhoria. Algumas seções ainda estão sendo aprimoradas para atingir conformidade completa.

Recursos de acessibilidade implementados

⌨️

Navegação por teclado

Todas as funcionalidades estão acessíveis via teclado. Indicadores visuais de foco (:focus-visible) estão presentes em todos os elementos interativos.

📢

Leitores de tela

O site é compatível com leitores de tela como NVDA, JAWS e VoiceOver. Utilizamos WAI-ARIA para comunicar estados e regiões de forma semântica.

🏷️

Rótulos e alternativas textuais

Imagens têm textos alternativos descritivos. Ícones decorativos são ocultados de tecnologias assistivas com aria-hidden.

📍

Estrutura semântica

Hierarquia de títulos (h1–h3) consistente em todas as páginas. Landmarks ARIA (main, nav, aside, footer) delimitam as regiões do conteúdo.

🔴

Regiões ao vivo

Atualizações dinâmicas de conteúdo (busca, carregamento de filmes, envio de formulários) são anunciadas via aria-live para usuários de leitores de tela.

🌈

Contraste de cores

Textos e elementos interativos atendem às proporções mínimas de contraste WCAG 2.1 AA (4,5:1 para texto normal, 3:1 para texto grande).

📋

Formulários acessíveis

Todos os campos de formulário têm rótulos (label) associados, mensagens de erro claras e suporte a autocomplete para facilitar o preenchimento.

🤟

Widget VLibras

Integração com o VLibras (ferramenta do Governo Federal) disponível em todas as páginas, oferecendo tradução automática de conteúdo para Libras.

Limitações conhecidas

Trabalhamos continuamente para melhorar, mas reconhecemos que algumas áreas ainda estão em aprimoramento:

  • Vídeos de trailers incorporados do YouTube podem não ter legendas em todos os casos — dependem da disponibilidade no próprio YouTube.
  • Conteúdo carregado dinamicamente (filmes em cartaz, sessões, relatos) pode ter um pequeno atraso antes de ser anunciado por leitores de tela.
  • O widget VLibras é uma ferramenta de terceiros — sua cobertura e precisão dependem do Governo Federal.

Tecnologias compatíveis

O site foi testado e é compatível com as seguintes combinações de navegador e tecnologia assistiva:

Chrome + NVDA (Windows)
Firefox + NVDA (Windows)
Safari + VoiceOver (macOS / iOS)
Chrome + TalkBack (Android)
Edge + Narrador (Windows)

Recomendamos o uso de navegadores e tecnologias assistivas lançados nos últimos 2 anos para melhor experiência.
